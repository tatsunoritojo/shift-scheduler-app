### 1. 接続部 A ストーリー化修正（最優先）

**現状の問題**: 連続線形スケール（1× → 120×）で「ただ拡大するだけ」になっている。発注側の体感は **「いきなりぶわっと広がって終わる」** で、設計意図（1 ヶ月の地形 → 04/14 火の詳細 → 機能深掘りへ）という物語が読み取れません。

**修正方針**: 連続スケール → **段階的解読**

- dwell（留まる瞬間）を 3〜4 箇所入れる
- 各 dwell で **コピーが入れ替わる**（読める瞬間を作る）
- dwell 間の推移は速く（加速度を変える、イージング調整）
- 各 dwell のテーマは発注側の想定：「月全体（macro）」→「04/14 火にズーム」→「この日の勤務可能設定」→「機能深掘りへ」

ただし **具体の dwell 数、scale 値、コピー文言、タイミング配分は ClaudeDesign の craft 判断に委ねます**。「連続」を「段階的解読」に変える、という方針だけ守れば OK。

スクロール距離（現 200vh）は維持で構いません。dwell を入れても同じ距離で収まるはず。もし物足りなければ 250vh まで伸ばしてよい。

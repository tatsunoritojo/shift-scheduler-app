シフト管理システム

管理者
管理者画面の使い方
  1. 設定（このタブ）: 開校時間と休校日を設定します
  2. シフト期間: シフトの募集期間を作成し、アルバイトに希望を提出してもらいます
  3. シフト構築: 提出された希望をもとにシフトを組み、事業主に承認申請します

開校時間設定

曜日ごとの開校時間を設定します。ここで設定した時間がシフトの基本枠になります。

開校時間の例外

祝日・長期休暇・特別営業日など、通常と異なる日を個別に設定します

新規シフト期間の作成

シフトを募集する期間を作成します。「募集中」にするとアルバイトがシフト希望を提出できるようになります。

アルバイトに表示される名前です
下書き = まだ非公開 / 募集中 = アルバイトに公開
この日時を過ぎると提出できなくなります（任意）

シフト構築

アルバイトが提出したシフト希望をもとに、実際のシフトスケジュールを組み立てます。カレンダーの日付をクリックして、スタッフの割当を行ってください。

スケジュール

「保存」で下書き保存、「承認申請」で事業主に確認を依頼、承認後に「確定・カレンダー同期」でGoogleカレンダーに反映します。

提出状況

アルバイトは期間中に何度でも再提出できます。日時は最終提出時刻です。

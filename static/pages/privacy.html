プライバシーポリシー

最終更新日: 2026年3月16日

シフリー（以下「本サービス」）は、東城立憲（以下「運営者」）が運営するシフト管理サービスです。本プライバシーポリシーは、本サービスにおける個人情報の取り扱いについて定めます。

1. 収集する情報

本サービスでは、以下の情報を収集します。

1.1 Googleアカウント情報

Google OAuth 2.0 を通じてログインする際に、以下の情報を取得します。

  • メールアドレス
  • 表示名
  • プロフィール画像URL

1.2 Googleカレンダー情報

シフト管理機能を提供するため、お客様の同意のもとに以下のアクセスを行います。

  • カレンダーの予定の読み取り（営業時間・休校日の同期）
  • カレンダーへの予定の書き込み（確定シフトの反映）

1.3 サービス利用情報

  • シフト希望データ（勤務可能日時）
  • 組織メンバーシップ情報（所属組織、ロール）

2. 情報の利用目的

収集した情報は、以下の目的で利用します。

  • ユーザー認証およびアカウント管理
  • シフト希望の収集・管理・表示
  • 確定シフトのGoogleカレンダーへの反映
  • 欠員補充リクエストの通知
  • サービスの運用・改善

3. 情報の共有

お客様の個人情報は、以下の場合を除き第三者に提供しません。

  • 同一組織内の管理者に対するシフト関連情報の共有（サービス機能として必要な範囲）
  • 法令に基づく開示要求があった場合
  • お客様の同意がある場合

4. Googleユーザーデータの取り扱い

本サービスは Google API サービスのユーザーデータポリシー（Limited Use requirements を含む）に準拠しています。

  • Googleから取得したデータは、本プライバシーポリシーに記載された目的にのみ使用します
  • Googleのユーザーデータを広告目的で使用しません
  • Googleのユーザーデータを第三者に販売しません
  • Googleのユーザーデータをセキュリティ目的以外で他のデータセットと結合しません

5. データの保管とセキュリティ

  • データはSSL/TLS暗号化通信により保護されています
  • パスワードは保存しません（Google OAuth 2.0 による認証のため）
  • Googleのリフレッシュトークンは暗号化して保管します

6. データの削除

お客様は以下の方法でデータの削除を要求できます。

  • 組織からの脱退：組織内のデータが削除されます
  • アカウント削除の要求：運営者へメールにてご連絡ください
  • Googleアカウントとの連携解除：Googleアカウントの権限管理から本サービスのアクセスを取り消せます

7. Cookieの使用

本サービスでは、セッション管理のためにCookieを使用します。これはサービスの正常な動作に必要なもので、トラッキングや広告目的には使用しません。

8. 変更について

本プライバシーポリシーは、必要に応じて変更することがあります。重要な変更がある場合は、サービス上で通知します。
